LineUp.

Projection-based AR Language Learning System, Learning a language by playing a sorting game.

LineUp helps children learn English grammar by playing word sorting games. This system consists of four toio™ Core Cubes, a portable projection device and the development mat. Children are encouraged to move cubes to play the sorting game; they'll have fun using the language learning process. It helps attract children’s attention and increase the passion of learning a foreign language. This project is part of the Student Innovation Competition at UIST 2021.

Word Sorting Game

Learning a language by playing a sorting game. Supported by projection AR, each cube will be assigned a part of a sentence, users could move the cube to rearrange the order of the sentence.

Technique specification

Refer to toio™ Core Cube Specifications, We can easily get the position of each cube through the identification sensor in the bottom of the cube. And the user only need to press the cube from the top, the button will be triggered at the same time.

When the system received the position of the cube, it will be calculated as a new position for the word, which is supposed to be attatched to the top of the cube by the projection.
